Para el desarrollo de las prácticas y proyectos se han utilizado diferentes herramientas relacionadas con los entornos de desarrollo y el control de versiones.



**- Visual Studio Code**: editor de código fuente ligero y muy utilizado para el desarrollo de aplicaciones en distintos lenguajes.



**- Apache NetBeans**: entorno de desarrollo integrado (IDE) orientado principalmente al desarrollo en Java.



**- Git**: sistema de control de versiones que permite gestionar los cambios en el código fuente.



**- GitHub**: plataforma online que permite almacenar repositorios Git y facilitar el trabajo colaborativo.
